LewisStructure
A small toy app I created during a college chemistry course. Performs basic(!) 3D visualization of molecules, using only a 2D graphics library, and attempts to calculate dipole moments.
BRSerker

A simple, web-based, build editor and generator for the brick building game Brickadia.

Primary purpose of this project is to create basic components of more complicated builds which would otherwise be tedious/difficult to make manually. Examples include large sections of randomly generated terrain, walls, and floors; as well as efficiently voxelized (representation in fewest possible bricks) images and 3D models.

C# Development
Tinker's Satchel (Risk of Rain 2)
A mod (third-party modification) for Risk of Rain 2. It adds significant amounts of original content to various areas of the game, including over 40 new items (base game has 186) with custom models.
TILER2 (Risk of Rain 2)
A library mod for Risk of Rain 2, which has only minimal relevance for players as it instead supports other mods. It provides:
  • a module system to assist in organizing complicated mods;
  • built-in module types for easy creation of new items, equipments, and artifacts;
  • a sophisticated layer over a popular config system, adding net sync, runtime updates incl. file watching, and one-line property binding;
  • a way to track and remove AI nav nodes;
  • and various other utilities.
Radial Pings (Risk of Rain 2)
A mod for Risk of Rain 2 which expands the base game's multiplayer ping system. Adds new ping types, accessible from a radial menu by holding the ping key down.
Admiral (Risk of Rain 2)

A mod for Risk of Rain 2 which overhauls the Captain character.

Features include reworked, non-permanent, reusable, and usable-everywhere Supply Beacons; primary attack autofire and rebalancing; major buffs to the Shocked status; and several new alternate skills (incl. deployable Jump Pads).
