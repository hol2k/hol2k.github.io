A Selection of WebDev Projects
Created as part of a web development training course in early 2022
Featured
CellAuto

A full-fledged, single-page website.

Includes a cellular automaton simulation engine, built on JavaScript, HTML Canvas, and HTML Forms. Also includes a short article explaining what cellular automata are.

Homework 1: Site Design

A full-featured, one-page, static website.

Includes HTML5 semantic tags, a rich header bar with logo and background, a sticky top navbar leading to other pages, a show-on-hover side fixed navbar that smoothly scrolls to articles within the page, and use of the FontAwesome icon library.

HTML5 & Better Navbar

A one-page, dynamic website, which uses HTML5 semantic tags to help automatic systems (e.g. search engines) understand the site's layout.

Also includes an expand-on-hover sticky navbar, as well as advanced use of CSS gradients to create an interesting site background image.

CSS3 Cube Spinner

A demonstration of CSS3's transform and transition properties, as well as some JavaScript, to create a 3D cube which can be rotated and resized via a control panel.

Animated Clock

Demonstrates use of JavaScript and HTML Canvas to create complicated animations. Also demonstrates use of JS Date/Time features.

JSON Searcher

A control panel to perform advanced searches of an internal, randomly-generated JSON database.

Self-Study
Number Format Converter

A widget that converts typed numbers between numeric and verbal formats, providing feedback to the user if either input is invalid.

RNG and WebGL

A demonstration of the THREE.js third party library, plus simple implementations of an Entity Component System and Perlin noise, to create a 3D rendering of procedurally-generated terrain.

Vector Plot

Made while studying the HTML Canvas element.

Creates vector plots of some predefined mathematical functions. Also includes a work-in-progress version of the other Voronoi plotter, which was later split off.

Input Diagram

Made while studying the HTML SVG feature, CSS animations, and JavaScript.

Visualizes keyboard keys as they're pressed, and animates them flying into place in a text field (or out of it, when backspace is pressed). Also allows mouse inputs.

Voronoi Diagram

Made while studying the HTML Canvas element and JavaScript.

A basic visualization of a random Voronoi diagram. Refresh to display a new diagram.

A one-page, dynamic website containing a full-featured job application form (albeit lacking a backend application to receive submissions).

Demonstrates form fields, fieldsets, field validation and requirements, and advanced CSS layout via the Flexbox feature; and use of the CSS clip-path property to create custom button shapes.
